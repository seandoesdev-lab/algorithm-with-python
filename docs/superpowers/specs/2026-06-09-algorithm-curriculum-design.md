# 파이썬 알고리즘·자료구조 학습 커리큘럼 — 설계 문서

- 작성일: 2026-06-09
- 목표: 파이썬 알고리즘 및 자료구조를 기초부터 심화까지, 그리고 대기업 공채 코딩테스트 기출까지 매일 한 단계씩 학습할 수 있는 콘텐츠 시스템 구축

## 1. 핵심 결정 사항 (확정)

| 항목 | 결정 |
|---|---|
| 콘텐츠 형식 | 마크다운(개념) + 파이썬 파일(코드) |
| 언어 | 한국어 설명 + 핵심 용어 영어 병기 (예: 시간복잡도(Time Complexity)) |
| 하루 단위 | 개념 + 예제 + 문제 세트 |
| 생산 방식 | Day by Day (매일 하나씩 점진적 생산) |
| 자동화 | Claude 스케줄 등록 — **매일 아침 8시** 다음 Day 자동 생성 |
| 진행 구조 | 1단계 개념 집중기(약 30일) → 2단계 문제 풀이기 |

## 2. 전체 진행 구조 (2단계)

### 1단계 — 개념 집중기 (Concept Sprint, 약 Day 1~30)

자료구조·핵심 알고리즘 개념을 **매우 자세하게** 다룬다.

- 개발자가 반드시 알아야 하는 내용은 문장이 길어져도 무조건 포함한다.
  (예: dict가 왜 평균 O(1)인가 — 해시 충돌·재해싱까지, 리스트 슬라이싱의 숨은 비용 등)
- 각 개념마다 **웹 서칭으로 수집한 직관·암기 팁·흔한 함정**을 별도 섹션으로 넣는다.
- 개념을 빠르고 집중적으로 끝내고 다음 단계로 넘어가는 것이 목표.

### 2단계 — 문제 풀이기 (Problem Drilling, Day 31~)

매일 **유형/카테고리 + 난이도별**로 문제를 풀이한다.

- 개념 정리가 끝난 시점부터는 **대기업 코딩테스트 기출 위주**로 구성해 실전 성장.
- 한 문제에 대해 **여러 접근 방식을 각각 코드 예시로** 제공 (브루트포스 → 최적화, 재귀 → DP 등, 각 방식의 복잡도 비교).

## 3. 전체 커리큘럼 로드맵

| Phase | 주제 | 일수(대략) | 다루는 내용 |
|---|---|---|---|
| 0. 파이썬 코테 기초 | Python for Coding Test | ~5일 | 빠른 입출력, 자료형, 리스트/딕셔너리/셋, 컴프리헨션, 함수, `collections`/`heapq`/`itertools` |
| 1. 자료구조 기초 | Data Structures | ~10일 | 배열·리스트, 문자열, 스택, 큐, 덱, 해시, 연결 리스트, 트리 기본 |
| 2. 알고리즘 기초 | Core Algorithms | ~8일 | 시간복잡도(Big-O), 정렬, 이분 탐색, 투 포인터, 슬라이딩 윈도우, 누적 합, 그리디, 재귀 |
| 3. 탐색·그래프 | Search & Graph | ~7일 | 완전 탐색, DFS/BFS, 백트래킹, 힙·우선순위 큐, 그래프 기초 |
| (이상 약 30일 = 1단계 개념 집중기) | | | |
| 4. 심화 알고리즘 | Advanced | 문제 풀이기에 녹여 진행 | DP, 최단 경로(Dijkstra/Floyd/Bellman-Ford), MST(Kruskal/Prim), 위상 정렬, Union-Find, 트라이, 세그먼트 트리, 비트마스킹 |
| 5. 대기업 코테 실전 | Real Coding Test | 지속 | 삼성 SW역량(구현·시뮬레이션), 카카오 블라인드, 라인/쿠팡 유형, 난이도별 기출 모의 세트 |

> Phase 4 심화 알고리즘은 별도 개념일과 기출 문제 풀이를 섞어 2단계에서 점진적으로 다룬다.

## 5. 하루치 콘텐츠 4종 파일

### concept.md — 개념 집중기 강화 구조
```
1. 정의와 직관 (Definition & Intuition)
2. 동작 원리 — 내부 동작까지 자세히
3. 시간/공간 복잡도 + 왜 그런지 설명
4. 파이썬 관용구 & 표준 라이브러리 사용법
5. 💡 이해를 돕는 팁 (웹 서칭 기반 — 비유, 암기법, 흔한 함정)
6. ⚠️ 개발자 필수 상식 (길어도 반드시 포함)
7. 언제 쓰고 언제 쓰면 안 되는가
```
> 2단계(문제 풀이기)에는 concept.md를 그날 다루는 유형의 핵심 패턴 요약으로 가볍게 운용.

### examples.py
개념을 보여주는 실행 가능한 예제 코드. 단계별 주석 포함.

### problems.md
- 출처: 백준, 프로그래머스, LeetCode, 삼성 SW역량테스트 기출 등 (사이트명 + 번호 + 링크 매핑)
- 난이도 분리: 🟢기초 / 🟡중급 / 🔴심화 / ⚫기출(대기업)
- 카테고리(유형)별로 분류, 힌트 포함

### solutions.py
- 접근 방법 설명 주석 + 정답 코드 + 시간복잡도 분석
- **코테 문제는 여러 접근 방식을 각각 코드 예시로 제시** (예: 브루트포스 → 최적화, 재귀 → DP), 각 방식의 복잡도 비교

## 6. 매일 자동 실행 (Claude 스케줄)

`schedule` 기능으로 **매일 아침 8시** 다음 Day 콘텐츠를 자동 생성하는 루틴을 등록한다.

루틴 동작 순서:
1. `curriculum.md`의 진도(다음 미완료 Day) 확인
2. 해당 Day 주제 파악
3. 웹 서칭으로 개념 팁/기출 문제 수집
4. 4종 파일(concept/examples/problems/solutions) 생성
5. `curriculum.md` 및 `README.md` 진도 체크리스트 갱신
