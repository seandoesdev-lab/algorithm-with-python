## 🟢 문제 1. 올바른 괄호
- 출처: 프로그래머스 #12909 — https://school.programmers.co.kr/learn/courses/30/lessons/12909
- 카테고리: 스택 / 괄호 매칭 (기초)
- 요약: `(`와 `)`로만 이루어진 문자열 `s`가 **올바른 괄호**이면 `True`, 아니면 `False`를 반환한다.
  예: `"()()"`, `"(())()"`는 올바르고, `")()("`, `"(()("`는 올바르지 않다.
- 💭 힌트: 카운터 하나로도 가능하다. `(`면 +1, `)`면 -1. 도중에 음수가 되면 실패, 끝에 0이 아니어도 실패.
  스택으로 풀면 `(`를 push, `)`에서 pop(비었으면 실패) -> 끝에 스택이 비어야 성공.

## 🟡 문제 2. Valid Parentheses (여러 종류 괄호)
- 출처: LeetCode #20 — https://leetcode.com/problems/valid-parentheses/
- 카테고리: 스택 / 괄호 매칭
- 요약: `()[]{}` 세 종류 괄호로 이루어진 문자열이 올바르게 **짝과 순서**가 맞는지 판별한다.
- 💭 힌트: 닫힘->열림 매핑 `{')':'(', ']':'[', '}':'{'}`. 여는 괄호는 push, 닫는 괄호는
  top과 매핑이 일치하는지 확인 후 pop. 비어 있는데 닫힘이 오거나, 끝에 남으면 실패.

## 🟡 문제 3. Min Stack (최솟값 O(1) 스택 설계)
- 출처: LeetCode #155 — https://leetcode.com/problems/min-stack/
- 카테고리: 스택 / 자료구조 설계
- 요약: `push`, `pop`, `top`, **`getMin`(현재 스택의 최솟값)** 을 **모두 O(1)** 로 지원하는
  스택을 설계한다.
- 💭 힌트: 값 스택과 별도로 **"그 시점까지의 최솟값"을 함께 쌓는 보조 스택**을 둔다.
  push 시 `min(x, 현재최소)`를 같이 쌓고, pop 시 둘 다 뺀다. (한 스택에 (값, 최소) 튜플로도 가능.)

## 🔴 문제 4. 주식가격
- 출처: 프로그래머스 #42584 — https://school.programmers.co.kr/learn/courses/30/lessons/42584
- 카테고리: 스택 / 모노토닉 스택 (Next Smaller)
- 요약: 초 단위 주식가격 배열 `prices`에서, 각 시점의 가격이 **떨어지지 않은 기간(초)** 을 구한다.
- 💭 힌트: 이중 루프면 O(n^2). **모노토닉 스택**으로 O(n): 인덱스를 스택에 쌓다가 현재 가격이
  스택 top의 가격보다 **낮아지면** pop하며 그 순간까지의 기간을 확정한다. 끝까지 안 떨어진 것은
  `n-1-i`로 마무리.

## 🔴 문제 5. Daily Temperatures
- 출처: LeetCode #739 — https://leetcode.com/problems/daily-temperatures/
- 카테고리: 스택 / 모노토닉 스택 (Next Greater)
- 요약: 온도 배열에서 각 날에 대해 **더 따뜻한 날까지 며칠** 기다려야 하는지(없으면 0) 반환한다.
- 💭 힌트: 인덱스를 쌓는 모노토닉 스택. 현재 온도가 스택 top의 온도보다 **높으면** pop하며
  `현재인덱스 - 그인덱스`를 답으로 기록. 각 원소가 최대 한 번 push/pop이라 O(n).

## ⚫ 문제 6. 크레인 인형뽑기 게임 (2019 카카오 개발자 겨울 인턴십)
- 출처: 프로그래머스 #64061 — https://school.programmers.co.kr/learn/courses/30/lessons/64061
- 카테고리: 스택 / 시뮬레이션 (기업 기출)
- 요약: `N x N` 격자 `board`에서 `moves`가 가리키는 열의 가장 위 인형을 차례로 집어 바구니에
  넣는다. 바구니 맨 위 인형과 **같은 인형**이 연속으로 들어오면 둘 다 터져 사라진다.
  터져 사라진 인형의 **총 개수**를 반환한다.
- 💭 힌트: 바구니를 **스택**으로 본다. 각 move에서 해당 열을 위->아래로 훑어 첫 인형(0이 아닌 값)을
  집고, 스택이 비어있지 않고 top과 같으면 `pop` 후 카운트 += 2, 다르면 `push`.
